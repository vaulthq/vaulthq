Vault? What?

Have you ever had your passwords stored in text file? Or on a sticky note? Delivered by email? Or were they on that Excel file which you have lost?

That kind of secret storage is insecure, slow and unreliable. Vault is a web based tool to solve password and other secret information storage and sharing problems.

Vault Overview

Security as #1 priority

Every secret in Vault is encrypted using RSA private-public key encryption. This ensures that data is not readable even if storage was breached.

Created for teams

Secrets are grouped by projects. It is possible to easily share all secrets in project, or if you do not want such loose control, you can also share them individually.

Powerful integrations

Vault provides an API. This allows external software to communicate with Vault, making integrations such as automated deployment tools, browser extensions possible.

Free & Open Source

What could be better than free, community driven software?
